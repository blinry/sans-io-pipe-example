# sans-IO pipe trait example

See [src/main.rs](src/main.rs) for an explanation.

You can run the example like this:

    cargo run

Or like this:

    seq 1 10 | cargo run

Or like this:

    echo -e "21\nfu\n5" > in
    cargo run <in >out
